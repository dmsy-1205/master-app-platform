    Routing Manager

    Beginner App Registration

    초보자용 앱 등록 가이드

    앱 파일을 apps/앱파일.html 형태로 넣고 아래 정보를 등록하면 App Store에 표시됩니다.

    1. 앱 HTML 파일을 apps 폴더에 넣습니다.
    2. 앱 이름만 입력하면 ID와 라우팅 경로는 자동 추천됩니다.
    3. Entry URL은 내부 앱이면 ./apps/파일명.html 외부 앱이면 https://주소를 입력합니다.
    아가 생활관리 앱 보안 배포 흐름 원본 생활관리 앱을 그대로 외부 공개하지 않고 apps/baby-care-secure.html 보안 진입 페이지로 먼저 연결합니다. 최종 보호는 Firebase Rules에서 승인 사용자만 rooms 데이터에 접근하도록 막는 방식으로 완성합니다.
    생활관리 앱 보호 체크리스트
    • 외부 Netlify 주소를 직접 공개하지 않기
    • Master App Platform 승인 사용자만 App Store에서 실행
    • 공유 코드 입력 방식은 추측 불가능한 코드로 교체
    • 다음 보안 단계에서 Firebase Rules 적용
    영문 소문자 숫자 하이픈 권장
