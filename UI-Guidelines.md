# UI Guidelines

- 사용자 화면은 App Store와 실행 경험 중심
- 관리자 화면은 정보 밀도와 빠른 제어 중심
- 위험 동작은 확인창을 거친다
- 생활관리 앱 배포 흐름은 초보자도 따라 할 수 있게 버튼과 안내문을 함께 제공한다


## STEP9-v8 UI 기준
Dashboard에는 핵심 앱만 노출하고 다수 앱 탐색은 App Store 그리드에서 처리한다. 즐겨찾기는 최대 3개까지 Dashboard에 요약 표시하고 전체 앱 관리는 App Store에서 진행한다.


## STEP10-v9 Login Gate UI 기준

- 공개 대문은 중앙 로그인 카드 하나를 기본으로 한다.
- 회원가입과 비밀번호 찾기는 별도 큰 카드가 아니라 같은 카드 내부 전환으로 제공한다.
- 상단 마케팅 메뉴는 제거하고 브랜드 로고만 유지한다.
- 버튼 색상은 기존 MasterOS 블루 퍼플 그라디언트를 사용한다.
- 모바일에서는 카드 폭을 줄이고 여백을 축소하되 입력창 터치 영역은 54px 이상 유지한다.
