# STEP8-8 Mobile Drawer Interaction & Compact Session Fix

## 적용 파일
- index.html
- workspace.js
- assets/css/platform-shell.css

## 적용 방법
1. 기존 프로젝트의 같은 경로에 위 파일을 덮어씁니다.
2. GitHub에 커밋/푸시합니다.
3. Netlify 배포 완료 후 Android/Samsung Chrome에서 강력 새로고침 또는 새 탭으로 확인합니다.

## 검수 항목
- 모바일 폭이 계속 100%로 유지되는지
- 메뉴 버튼을 누르면 메뉴가 열리는지
- 메뉴가 열린 상태에서 메뉴 버튼을 다시 누르면 닫히는지
- 메뉴 항목을 누르면 메뉴가 닫히고 화면 이동하는지
- 일반 사용자에게 Admin Center가 보이지 않는지
- Home 상단 상태 카드가 다시 나타나지 않는지
