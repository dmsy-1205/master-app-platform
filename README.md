# Master App Platform

현재 버전: `master-app-platform-step08-v3`

## 개요
Master App Platform은 여러 웹 애플리케이션을 하나의 플랫폼에서 등록 관리 실행하기 위한 통합 플랫폼입니다.

## 현재 완료
- 회원가입 로그인 로그아웃
- Firebase Realtime Database CRUD
- 관리자 권한 시스템
- 회원 신청 및 승인 시스템
- 다중 서브 애플리케이션 등록
- 동적 라우팅
- 사용자 Dashboard
- 앱 실행 연결
- App Store형 UI 구조 개편

## STEP8-v3 변경점
- 로그인 전 화면을 전문 플랫폼 랜딩 페이지로 개선
- 회원가입/로그인 UI 개선
- 사용자 Workspace 구조 적용
- App Store 카드형 앱 목록 적용
- 관리자 센터 분리
- 개발 테스트 도구 관리자 전용 유지

## 배포 방식
ZIP 압축 해제 후 GitHub에 덮어쓰기 업로드하면 Netlify 자동 배포로 테스트합니다.
