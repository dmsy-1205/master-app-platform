# MasterOS Modules

STEP12부터 기능별 모듈 경계를 문서화한다.
현재 배포 안정성을 위해 기존 JS 파일은 유지하고 다음 단계에서 점진적으로 내부 코드를 이동한다.

- auth: 로그인 회원가입 비밀번호 찾기 세션
- workspace: 좌측 메뉴 라우팅 화면 전환
- store: App Store 신청 실행 상세보기
- runtime: Launch Token Runtime Execution Log
- admin: 회원 승인 앱 등록 버전 관리 QA
- feedback: 요청 게시판 관리자 답변
- notifications: 알림 센터 공지
