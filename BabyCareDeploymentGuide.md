# 아가 생활관리 앱 보안 배포 가이드

## 현재 위험

외부 Netlify 주소에 생활관리 앱을 그대로 배포하면 URL을 아는 사람이 앱 화면과 클라이언트 코드에 접근할 수 있습니다. Firebase 설정값과 Realtime Database 경로도 브라우저 코드에서 확인될 수 있습니다.

## 권장 구조

1. 생활관리 앱 원본을 외부 공개 URL 단독 배포로 두지 않습니다.
2. Master App Platform에 승인 사용자 계정으로 로그인합니다.
3. App Store에서 `아가 생활관리 앱`을 실행합니다.
4. 앱 데이터는 Firebase Rules로 승인 사용자만 접근하게 제한합니다.
5. 공유 코드 방식은 추측하기 어려운 코드 또는 사용자별 권한 매핑으로 교체합니다.

## STEP9-v7 적용 내용

- 관리자 앱 등록 프리셋을 보안 진입점으로 변경
