# STEP8-1 Navigation Session & Global Contrast Fix

## 적용 파일
- index.html
- auth.js
- dashboard.js
- assets/css/platform-shell.css

## 적용 방법
현재 GitHub 프로젝트 루트에 위 파일들을 같은 경로로 덮어씁니다.

## 배포 후 확인
1. PC 화면에서 상단 로그인 정보 카드가 사라지고 좌측 메뉴 안에 로그인/로그아웃 정보가 표시되는지 확인합니다.
2. 로그인 상태에서 UID가 화면에 노출되지 않는지 확인합니다.
3. 활동 기록의 “생활관리 앱” 글자가 배경에 묻히지 않고 읽히는지 확인합니다.
4. 앱 둘러보기, 앱 신청, 공지사항, 고객지원, 관리자 센터에서 흐린 글자/회색 박스가 줄어들었는지 확인합니다.
5. 모바일/Fold에서 메뉴 버튼과 좌측 드로어가 정상 작동하는지 확인합니다.
6. 앱 실행, 승인, Firebase 권한, 활동 기록 저장이 기존처럼 작동하는지 확인합니다.

## 수정하지 않은 영역
Firebase Rules, 앱 실행 보안 검증, 승인 데이터 구조, Admin Center 데이터 로직은 수정하지 않았습니다.
