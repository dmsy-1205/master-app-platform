# STEP8-7 Stability Cleanup 적용 안내

## 목적
STEP8 계열 수정이 누적되며 발생한 모바일 폭 / 모바일 메뉴 / 상단 상태 카드 / 권한 메뉴 표시 문제를 안정화합니다.

## 적용 파일
- index.html
- workspace.js
- assets/css/platform-shell.css

## 적용 방법
1. 현재 GitHub 프로젝트 루트에 포함 파일을 그대로 덮어씁니다.
2. GitHub에 커밋/푸시합니다.
3. Netlify 배포가 완료된 뒤 브라우저 캐시를 새로고침합니다.

## 검수 순서
1. PC 관리자 로그인: 좌측 메뉴, Admin Center 표시 확인
2. PC 일반 사용자 로그인: Admin Center 메뉴 숨김 확인
3. 모바일 일반 사용자: 화면 폭 정상, 오른쪽 빈 공간 없음 확인
4. 모바일 메뉴 버튼: 열기/닫기/메뉴 이동 확인
5. Home 상단 상태 카드: 일반 사용자/승인완료/로그인확인 큰 카드 미노출 확인
6. 앱 실행: 승인된 앱 실행 정상 확인

## 변경하지 않은 영역
Firebase Rules, 로그인, 승인, 앱 실행 보안, 데이터 저장 로직은 변경하지 않았습니다.
