# STEP8 적용 안내

## STEP8 Twitter-like Platform Frame Renewal

### 적용 파일
- index.html
- workspace.js
- assets/css/platform-shell.css

### 적용 방법
1. 기존 프로젝트에서 위 3개 파일을 같은 경로에 덮어씁니다.
2. GitHub에 커밋/푸시합니다.
3. Netlify 배포 완료 후 강력 새로고침으로 확인합니다.

### 검수
- PC: 좌측 고정 메뉴 + 중앙 프레임 구조 확인
- PC: 콘텐츠가 화면 끝까지 퍼지지 않고 적정 폭 안에서 표시되는지 확인
- Mobile/Fold: 메뉴 버튼을 눌렀을 때 왼쪽 드로어 메뉴가 열리고 닫히는지 확인
- 모든 화면: 회색/어두운 박스와 흰 글자 묻힘 현상 확인
- 앱 실행/승인/Firebase 로직 정상 유지 확인
