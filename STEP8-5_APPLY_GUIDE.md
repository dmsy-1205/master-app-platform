# STEP8-5 Mobile Menu Interaction & Hero Status Simplification

## 적용 파일
- index.html
- workspace.js
- assets/css/platform-shell.css

## 적용 방법
1. 기존 프로젝트에서 위 3개 파일을 같은 경로에 덮어씁니다.
2. GitHub에 커밋/푸시합니다.
3. Netlify 배포 완료 후 모바일 Chrome에서 강력 새로고침 또는 캐시 삭제 후 확인합니다.

## 검수
- 모바일에서 메뉴 버튼 터치 시 열림/닫힘 확인
- 메뉴 바깥 영역 터치 시 닫힘 확인
- 메뉴 항목 선택 시 닫힘 확인
- Home 상단의 권한/승인/세션 큰 카드가 사라졌는지 확인
- 일반 사용자에게 Admin Center가 보이지 않는지 확인
- PC 화면에서 좌측 메뉴와 본문이 정상 유지되는지 확인
